About

キーワードと出版年をもとに本を検索することができます。

書誌情報とあわせて、書影・内容紹介・所蔵機関数・レビュー平均点・レビュー件数を表示することでページを行き来せずに本を探せることを目指しています。

Data Source

以下のAPIを使用しています

  • CiNii Books : 書誌情報
  • NDL Search : 書影（JPRO）
  • OpenBD : 内容紹介
  • Rakuten Books : レビュー

以下のオープンデータを使用しています

  • NDLSH(TAB区切りテキスト形式データ)
